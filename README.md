A dynamic and animated weather dashboard that displays real-time weather data using the OpenWeather API.

## Features
- Live weather data by city
- Animated backgrounds based on weather conditions
- Snow, rain, haze, clouds, and thunder effects
- Day / night detection
- Clean and responsive UI

## Tech Stack
- HTML
- CSS
- JavaScript
- OpenWeather API

## How to Run
1. Clone the repository
2. Open `index.html` in a browser
3. Enter a city name to view weather details
